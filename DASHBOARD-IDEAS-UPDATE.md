# Dashboard and fantasy polish update

This version adds the next round of app polish:

- Logged-in homepage is more compact.
- Main action buttons sit higher and horizontally.
- Homepage now shows a next-step status prompt.
- Homepage saved-team cards now show C / VC badges.
- C / VC badges update automatically because they are read from the currently open event's saved fantasy team.
- Pick Team page now includes a team completeness checklist.
- My Team page has clearer event status labels: Open, Locked, Scored, Missed and Upcoming.
- Public leaderboard now shows popular picks, captain picks, vice-captain picks and differential labels for selected event leaderboards.
- Mobile users get a sticky bottom navigation bar.
- Admin links are labelled more clearly as Admin Stats / Website statistics.
